## For Raspberry Pi,
- `npm install raspi-io`

## For Arduino Nano,
- Arduino IDE download
- File > Example > Firmata > StandardFirmataPlus
- Tools > Board > Arduino Nano
- Tools> Processor > ATmega328P (Old Bootloader)
- Tools> Port > usbserial-xxx
